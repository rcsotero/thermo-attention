# The Thermodynamics of Attention: First Law and Landauer Limit Analogues for Learning and Explainability

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

This codebase provides the tools to model Transformer self-attention via non-equilibrium statistical mechanics. Use these scripts to replicate the neural Landauer limit experiments, track the First Law of Learning, map attention phase diagrams, and evaluate energy-based feature attribution (XAI) across Vision Transformer (ViT) architectures.
